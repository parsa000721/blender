# Blender-Lite (Prototype)

## Run locally
1. npm install
2. npm run dev
3. Open http://localhost:3000

## Notes
- This is a lightweight Next.js + Three.js prototype.
- API routes use ephemeral storage (/tmp) for demo purposes. For production replace with S3 or other cloud storage and a database for projects.
- To deploy to Vercel: push this repo to GitHub and connect the repo on Vercel.
